# Interactive Grocery Store Web Application

# Project Overview
This project is a front-end web-based grocery store application developed using HTML, CSS, and JavaScript. It simulates an online shopping experience where users can browse products, add items to a cart, and interact with different sections of the website.

# Problem Statement
In modern daily life, managing grocery shopping efficiently can be challenging due to time constraints and lack of organization. Traditional shopping methods do not provide a structured way to track selected items or quickly browse available products. 

This project addresses this issue by providing a simple and interactive digital solution that allows users to explore products and manage their shopping process efficiently.

# Objectives
- To design a user-friendly grocery shopping interface  
- To implement dynamic cart functionality using JavaScript  
- To demonstrate event-driven programming concepts  
- To create a responsive layout for multiple devices  
- To improve user interaction through smooth UI behavior  

# Features
- Product browsing using sliders  
- Add to cart functionality  
- Dynamic cart display using JavaScript  
- Search bar toggle  
- Login form UI  
- Responsive navigation menu  
- Customer reviews section  
- Blog section with external resources  

# System Logic
The application follows an event-driven approach:

- User actions (clicks) trigger JavaScript functions  
- Products are stored in an array (cart system)  
- DOM manipulation updates the cart dynamically  
- UI components are controlled using class toggling  

# Workflow
1. User opens the website  
2. Browses products  
3. Clicks "Add to Cart"  
4. Product is stored in an array  
5. Cart updates dynamically  
6. User views selected items  

# Technologies Used
- HTML (Structure)
- CSS (Styling & Responsiveness)
- JavaScript (Logic & Interactivity)
- Swiper.js (Product sliders)

# Responsiveness
The website is fully responsive and adapts to:
- Desktop screens  
- Tablets  
- Mobile devices
